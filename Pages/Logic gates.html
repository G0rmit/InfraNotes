There are 5 types of logic gates:

  1. AND gate
  2. OR gate
  3. NOT gate
  4. XOR gate
  5. NAND gate

--> AND gate gives TRUE statement only if both inputs are TRUE.

--> OR gate gives TRUE statement if al least one input is TRUE.

--> NOT gate gives opposite from input statement. For example, if input is TRUE(or 1), then the output will be FALSE(or 0).

--> XOR gate gives TRUE output when only one input is TRUE. When both inputs are same, 0 0 or 1 1, it gives FALSE.

--> NAND gate is a mix of AND and OR gates. It acts opposite to AND gate: gives TRUE output in all cases, except when both inputs are TRUE.
