Memory vs Storage

Memory is short-term, used for running programs, while storage is long-term, keeping data even when computer is turned off.

HHD and SSD

Hard Drive Disk uses magnetic storage and have moving parts inside, making it slower and replacable faster. Solid State Drive uses flas memory, they ae faster and more durable.

DRAM

DRAM is a type of volatile memory used in RAM.It stores data in a capacitor and must me refreshed thousands of times per second. It is cheaper than SRAM but slower.

SDRAM

It's a type of DRAM that is syncronized with CPU clock. In new versions there is Double Data Rate(DDR), which transfers data on both rising and falling edges of the clock.

Generations of DDR:

  • DDR2
  • DDR3
  • DDR4
  • DDR5

SRAM

SRAM is a type of volatile memory tat stores data using circuit of transistors, so it doesn't need to be refreched as DRAM.

ROM

ROM is a read only memory, meaning it's read-only. It is used to permanently store data and instructions that don't change.
