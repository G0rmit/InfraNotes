Command bar (Git Bash) commands:

home
  • mkdir - create a folder
  • touch name.txt - create a text file
  • cd/directory_name/ - go to directory
  • cd - return from directory
  • git init - create repo
  • git status - check documents before commit
  • git commit -m "Message" - commit (save) files with a message
  • git remote add origin "link to repo" - link local repo to could repo on github/gitlab
  • git push -u origin main/master - push (upload) files to the cloud to the branch main/master
  • git pull origin main/master - pull (download) files from the cloud to local storage/repo
