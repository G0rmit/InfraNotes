Networking and Security

home

Network consists of multiple devices, called hosts, that are connected to eac other and can communicate with one another and share resources.

Network Interface Card

NIC is a hardware that allows a device to connect to a network. It can be wired or wireless.

Network types

  • Local Area Network(LAN) covers a small area like home
  • Wide Area Network(WAN) connects multiple LAN in a large area like between cities
  • The Internet is a global network of networks

Modem is a device that connects your home to the ISP by converting dgital signals to analogs.

Router is a device that connects different networks and forwards data packets to their destination.

Switch is a device that connects devices within the same local network.

Data transmission

Data is broken into small chunks called packets, that contain metadata and content. To be able to send packets faster and safer, it uses packet switching.

IP adress is a unique device id. Before we were using ipv4, that has 32 bits, but now we switched to ipv6 with 128 bits.

tcp/ip 4-layer model

  • Application: formats the message, like http
  • Transport: splits data and makes sure it goes safely by using ports
  • Internet: handles addressing and routing, like ip address
  • Link: transmits data over physical cables or wifi

Cybersecurity

  • Phishing: pretending to be an official source to steal data(passwords)
  • Malware: software like viruses or spyware
  • Social Engineering: manipulating people into reveaing their information over a call or messaging
  • Password Attaks: using brute force to check for every combination or credential stuffing by using passwords from data leaks
