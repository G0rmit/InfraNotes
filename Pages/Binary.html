To transform from binary to hexa or octa, we take each bit in those systems and transform that number into binary representations of it. For example:

Lets transfer decimal to binary: 27 = 11011, because 16*1+8*1+4*0+2*1+1*1.

Octal:

In octal, each bit ranges from 0-7. Also, each bit in octal system equals 3 bits in binary system.

(203)8 = 3*1 + 0*8 + 2*64 = (131)10.

Hexadecimal:

Same as octal, but each bit goes up to 15. But since we can use only 0-9 numbers, we use letters:

A - 10, B - 11, C - 12, D - 13, E - 14, F - 15. Also, each hexadecimal bit takes 4 binary bits.

(203)16 = 3*1 + 0*16 + 2*256 = (515)10.


How to transfer octal and hexadecimal to binary and reverse:

(457)8 = (100101111)2 because each bit in octal equals 3 bits in binary: 4 = 100, 5 = 101, 7 = 111.

(3E9)16 = (001111101001)2 because each bit from hexa equals 4 bits from binaly: 3 = 0011, E(14) = 1110, 9 = 1001.

Negative binary:

If the number is negative, we inverse each bit(1 to 0, 0 to 1), and add 1 to the last, smallest, bit.

Example: -011000111 = 100111000 + 1 = 100111001
