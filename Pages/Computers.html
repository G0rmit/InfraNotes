Computers

Computer is a device that automatically manipulates information with a precise set of rules.

A general-purpose computer is a device that can execute an arbitrary set of instructions.

Hardware refers to the physical components of a computer, and software - set of commands and programs

Hardware of computer:
CPU
Central processing unit, where all calculations happen
RAM
Random access memory, where running programs live
Motherboard
Central device, that connects everything else into one system
Storage
Long term storage
Input devices
Ones that take information into system: keyboard, microphone, mouse
Output devices
Ones that show the output of processes: monitor, speaker
